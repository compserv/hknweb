CSV file format

Please upload a CSV file with the following requirements:

  • File must be semicolon (;) separated
  • First row must contain column headers
  • Required columns:
    • Bytes: comma (,) separated list of usernames
    • Bits: comma (,) separated list of usernames
    • Year
    • Semester: either "Spring" or "Fall"
